1. Station
  2. Station Link Creation Rule

  1. - To create first station on newly created line previous and next station can be blank.

  2. - Previous station can be blank if the station to be created is before first station. In this case next station should be selected and should not be left blank.

  3. - Next station can be blank if the station to be created after last station In this case previous station should be selected and should not be left blank.

  4. - To add station in middle of station, previous and next station cannot be blank and same. Previous and next station should be adjacent to each other or else error will occur.

  5. - To assign more than one line to a station, current station should be a junction.
